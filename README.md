# API REST Segura en ASP.NET Core

## Descripción
API desarrollada en ASP.NET Core aplicando autenticación JWT,
roles y arquitectura limpia.

## Tecnologías
- ASP.NET Core 8
- SQL Server
- JWT
- Swagger

## Arquitectura
- Clean Architecture
- Separación por capas
- Principios SOLID

## Seguridad
- Autenticación JWT
- Autorización por roles
- Validación de inputs

## Cómo ejecutar
1. Clonar repositorio
2. Configurar connection string
3. Ejecutar `dotnet run`
